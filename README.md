# Sky Reaper

**Sky Reaper** is a top-down arcade shooter built using **HTML, CSS, and JavaScript**, featuring fast-paced gameplay, boss battles, and dynamic difficulty scaling — all running directly in the browser.

---

## 🎮 Features

* ⚔️ Smooth shooting and movement
* 👾 4 enemy types with unique attacks
* 💀 Boss battles with multiple patterns
* ⚡ Power-ups (Shield, Life, Power, Ammo, V-Shot)
* 🔥 Combo scoring system
* 📈 Progressive difficulty scaling
* 🔊 Music + procedural sound effects
* 🎨 Clean UI with HUD and pause system

---

## 🛠️ Tech Stack

* HTML5 Canvas
* Vanilla JavaScript (ES6)
* CSS3
* Web Audio API

---

## 📁 Structure

```bash
assets/    # Images, sounds
css/       # Styles
js/        # Game logic (player, enemies, boss, etc.)
index.html
```

---

## 🎯 Controls

* **W / A / S / D** → Move
* **Arrow Keys** → Move
* **Space / Enter** → Shoot
* **Pause Button** → Pause

---

## 🚀 Run Locally

Just open:

```bash
index.html
```
